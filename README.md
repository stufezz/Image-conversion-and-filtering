# Image Conversion and Filtering

Небольшой учебный проект по цифровой обработке изображений на Python с использованием `OpenCV`, `NumPy` и `Matplotlib`.

Скрипт `lab.py` берет исходное изображение `image.jpg`, выполняет набор преобразований и сохраняет готовые результаты в отдельные файлы.

## Что делает программа

После запуска скрипт последовательно выполняет:

1. перевод исходного изображения в градации серого;
2. логарифмическое преобразование яркости;
3. степенное преобразование с разными значениями гаммы;
4. добавление шума типа "соль и перец";
5. медианную фильтрацию зашумленного изображения;
6. повышение резкости с помощью ядра свертки;
7. выделение границ операторами Робертса, Превитта и Собеля;
8. нерезкое маскирование;
9. удаление узкой помехи медианным фильтром.

## Какие файлы создает

После выполнения рядом со скриптом сохраняются результаты:

- `image_g.jpg` — полутоновое изображение;
- `log_img.jpg` — логарифмическое преобразование;
- `power_01.jpg`, `power_045.jpg`, `power_5.jpg` — степенные преобразования;
- `salt_pepper.jpg` — изображение с шумом;
- `hist_salt_pepper.jpg` — гистограмма шумного изображения;
- `filtered_median_3.png`, `filtered_median_9.png`, `filtered_median_15.png` — результаты медианной фильтрации;
- `sharpened.jpg` — повышение резкости;
- `edge_roberts.jpg`, `edge_prewitt.jpg`, `edge_sobel.jpg` — выделение границ;
- `unsharp.jpg` — нерезкое маскирование;
- `comparison_sharpening.jpg`, `comparison_plot.jpg` — сравнение методов повышения резкости;
- `strip_image.jpg`, `filtered_strip.jpg` — изображение с полосовой помехой и результат фильтрации.

## Как запустить

### 1. Установить зависимости

```bash
pip install numpy opencv-python matplotlib
```

### 2. Подготовить входное изображение

Убедитесь, что файл `image.jpg` лежит в корне проекта.

### 3. Запустить скрипт

```bash
python lab.py
```

## Структура проекта

- `lab.py` — основной скрипт обработки;
- `image.jpg` — входное изображение;
- `README.md` — описание проекта;
- выходные `.jpg` и `.png` файлы — результаты обработки.

## Примечание

Скрипт сохраняет файлы в текущую папку и в конце показывает окно `Matplotlib` с итоговым сравнением методов повышения резкости.
